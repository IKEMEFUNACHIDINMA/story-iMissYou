# iMissYou Animation

A simple web animation that tells a sweet story of a girl stick figure saying "I miss you" to a boy stick figure. The animation concludes with them sharing a hug and a heart text floating on the screen. This project is built using HTML, CSS, and JavaScript.

## Demo

[Live Demo](https://stick-story.vercel.app/) 

## Features

- Stick figure animations
- Text animations
- Responsive design
- Simple and clean code

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:
   
bash
   git clone https://github.com/IKEMEFUNACHIDINMA/story-iMissYou.git
   

2. Navigate to the project directory:
   
bash
   cd story-iMissYou
   

3. Open index.html in your web browser to view the animation.

## Usage

Simply open the index.html file in your browser to see the animation in action.
Send to your loved ones as you wish!
